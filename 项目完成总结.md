## 🚀 快速启动

### 方法一：使用启动脚本（推荐）

```bash
# 终端1 - 启动后端
./start-backend.sh

# 终端2 - 启动前端  
./start-frontend.sh

# 浏览器访问
http://localhost:5173
```

### 方法二：手动启动

```bash
# 启动后端
cd backend
go mod tidy
go run main.go

# 启动前端（新终端）
cd frontend
npm install
npm run dev
```

## ⚙️ 配置要求

### 已完成配置

**✅ 已对接官方API，开箱即用！**

系统已配置官方彩票数据API，无需额外配置：

```go
API: APIConfig{
    // 双色球官方API（中国福利彩票）
    ShuangseqiuURL: "https://www.cwl.gov.cn/cwl_admin/front/cwlkj/search/kjxx/findDrawNotice",
    // 大乐透官方API（中国体育彩票）
    DaletouURL: "https://webapi.sporttery.cn/gateway/lottery/getHistoryPageListV1.qry",
}
```

### API特点

**双色球官方API：**
- 真实的中国福利彩票官方数据接口
- 支持按期号查询
- 返回完整的开奖信息和奖金数据

**大乐透官方API：**
- 真实的中国体育彩票官方数据接口
- 支持按期号查询
- 返回完整的开奖信息和销售数据

这些都是公开的官方接口，可以直接使用，无需认证！
